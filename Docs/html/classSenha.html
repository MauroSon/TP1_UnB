Descrição detalhada

Classe para representar uma Senha de cinco dígitos.

Regras de Formato:

  • Cinco dígitos (0-9).
  • Não há dígito duplicado.
  • Os cinco dígitos não podem estar em ordem crescente nem decrescente.
